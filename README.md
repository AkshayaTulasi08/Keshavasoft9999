# KeshavSoft Internship – Task 2

## Objective
Recreated Task-1 website using a templating engine and a task runner.

## Technologies Used
- Templating Engine: Nunjucks
- Task Runner: Gulp
- Styling: Bootstrap 5 (CDN)

## Folder Structure
- src/templates/pages – page templates
- src/templates/partials – reusable components
- dist – final compiled HTML output

## How to Run
1. Install dependencies:

```bash
npm install
```

2. Run in development (hot-reload):

```bash
npm start
# or
npm run dev
```

3. Build for production and preview the built site locally:

```bash
npm run build
npm run preview
```

4. Or open `dist/index.html` after `npm run build`.

## Output
Final HTML files are generated inside the dist folder.
